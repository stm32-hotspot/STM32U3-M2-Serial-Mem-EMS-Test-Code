## Contributing guide

Although we value the community's feedback, please note that **neither** issues **nor** pull requests submitted to this repository will be **addressed or acted upon**. This module is provided **as-is** and is **not necessarily supported** by STMicroelectronics.

For any questions or feedback, please submit them to the ST Community in the appropriate [forum](https://community.st.com/t5/product-forums/ct-p/product-forums).

To report a **vulnerability**, please refer to the [SECURITY.md](./SECURITY.md) file for instructions.
